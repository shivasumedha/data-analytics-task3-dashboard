# 📊 Task 3: Interactive Dashboard (Power BI)

## 📌 Overview
This project is part of my Data Analytics Internship.  
In this task, I created an interactive dashboard to visualize key insights from the dataset.

---

## 🎯 Objective
- To convert data into visual insights  
- To build an interactive dashboard  
- To make data easy to understand using charts  

---

## 📊 Dashboard Features
- Bar Chart (Category-wise analysis)  
- Line Chart (Trend over time)  
- Pie Chart (Distribution of data)  
- KPI Cards (Total values/metrics)  
- Filters/Slicers for interactivity
